# Shared Materials

Place class notes, lab exercises, and other teaching materials in this directory.

When you run `playbooks/05_copy_shared_materials.yml`, the contents of this folder are copied to the student home directories under `Lab_Materials/`.

Example layout:

- `shared_materials/lecture1.txt`
- `shared_materials/lab-exercise-01.pdf`
- `shared_materials/setup-instructions.md`
